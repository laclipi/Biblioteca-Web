# Biblioteca-Web
Aplicación web para la gestión de libros, usuarios y préstamos.

## Tecnologías
- HTML
- CSS
- JavaScript (Programación Orientada a Objetos)

## Funcionalidades
- Registro de libros y usuarios
- Gestión de préstamos y devoluciones
- Control de disponibilidad de libros

## Estado
Proyecto funcional.

## Aprendizajes
- Uso de clases y objetos en JavaScript
- Modelado de lógica de negocio
- Desarrollo de interfaces web funcionales
