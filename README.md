# Biblioteca Web

Proyecto web para la gestión de una biblioteca.

## Descripción
Aplicación web para la gestión de libros, usuarios y préstamos.

## Tecnologías
- HTML
- CSS
- JavaScript (Programación Orientada a Objetos)

## Funcionalidades
- Registro de libros y usuarios
- Gestión de préstamos y devoluciones

## Estado
Proyecto académico / funcionando
